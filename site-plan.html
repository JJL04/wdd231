HealthyHabits Hub – Site Plan

1. Site Name

HealthyHabits Hub – This name reflects a community resource for individuals seeking to build healthy routines. The word "hub" conveys a central place for tools, articles, and progress tracking.

Optional domain: healthyhabits-hub.org

2. Site Purpose

This site provides wellness tips, articles, meal plans, and habit tracking tools for individuals striving for a balanced lifestyle. It also includes a community board for accountability and encouragement.

3. Scenarios

  • How can I start a healthy morning routine with limited time?
  • Where can I find beginner-friendly workout or meal plans?

4. Color Schema

  • Primary Color (#2c7a7b): used for headers, links, and buttons.
  • Accent Color (#d9534f): used for alerts, highlights, and CTA buttons.

5. Color Schema Use in Document

This HTML document uses the primary color for headings and the accent color for emphasis/highlighted text.

6. Typography

The main font is Montserrat from Google Fonts. It is used for all headings and body text, offering a clean and modern feel that matches the wellness theme.
